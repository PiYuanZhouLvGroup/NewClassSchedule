## 简单说明

有三种UI：正常UI、简化UI、极简UI

正常UI下 双击进度条 或 试图最小化窗口 时进入简化UI

简化UI 双击进度条 恢复正常UI，在进度条上中键（按滚轮）切换横/纵模式

正常UI下 使用斜杠指令`/xsbj`（或`/xsbj 时长`） 进入极简UI

其他的不多说，可以看源码，但是没有注释 ;)
